# Security Policy

## Supported Versions

We do not provide any support for previous versions of our 'software'.
By 'previous versions', this includes **ANY** version that does not run on our servers.

| Version | Supported          |
| ------- | ------------------ |
| Deployment   | :white_check_mark: |
| Source code   | :x:                |
